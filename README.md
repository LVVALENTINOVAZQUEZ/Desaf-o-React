# Challenge React - BotFilter

Proyecto desarrollado para el desafío técnico de BotFilter. La aplicación permite visualizar vacantes laborales y postularse enviando la URL del repositorio.

## Tecnologías utilizadas

- React.js (Vite)
- Fetch API para integración con servicios externos
- CSS-in-JS para estilos básicos

## Funcionalidades

1. Consumo de datos de candidato mediante email.
2. Listado dinámico de vacantes desde la API.
3. Formulario de postulación con validación de estados (Cargando/Éxito/Error).
